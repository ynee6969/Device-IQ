# Dialed CSS and HTML Practice Questions

## How To Use This File

- Practice answering each question in your own words
- Be ready to point to the exact HTML or CSS pattern in the project
- Focus on why the choice was made, not only how it works

---

## Semantic HTML

1. Why is it better to use `<header>`, `<main>`, and `<footer>` instead of only `<div>` elements?
2. Why should the navigation area use `<nav>`?
3. Why is each phone card better represented as an `<article>`?
4. Why is the compare section easier to understand when it is wrapped in meaningful semantic sections?
5. How does semantic HTML help screen readers understand the page?
6. Why should form fields use `<label>` elements instead of placeholder text alone?
7. Why is the page title important for both HTML structure and user experience?
8. Why should images include descriptive `alt` text?
9. Why is it helpful to separate page content into smaller semantic sections?
10. Why is semantic structure easier to defend in a presentation than random nested `<div>`s?
