# Dialed PowerPoint Guide

## Suggested 5-Slide Deck

### Slide 1: Title
- Dialed
- Phone discovery and comparison website
- Your name, section, date

**What to say:**
- Introduce the project in one sentence
- Say the website helps people choose phones more easily

### Slide 2: Problem and Purpose
- Too many phone choices
- Specs are scattered across different sites
- Users need a simpler way to decide

**What to say:**
- Explain the real-world problem first
- Emphasize that the website saves time and reduces confusion

### Slide 3: How The Website Helps
- Browse phones
- Compare devices side by side
- Save favorites for later
- Sign in only for account-based features

**What to say:**
- Walk through the user journey
- Mention that browsing stays open and easy

### Slide 4: Design Rationale
- Clean and modern visual style
- Strong contrast and readable layout
- Responsive on mobile and desktop
- Clear cards and spacing for easier scanning

**What to say:**
- Explain that the design is meant to support clarity
- Mention that the interface should feel premium but easy to use

### Slide 5: Closing
- Dialed simplifies phone selection
- It organizes comparison and saving in one place
- It is built for speed, clarity, and usefulness

**What to say:**
- End with the main rationale
- Invite questions

---

## Slide Order Tips

- Keep text short on each slide
- Use screenshots of the website instead of long paragraphs
- One idea per slide is better than crowded slides
- Show the browsing, compare, and signup pages in that order

## Visual Suggestions

- Use full-width screenshots of the website
- Add callout arrows for key features
- Keep the color palette close to the site itself
- Avoid too much text on the screen

## Speaker Strategy

- Talk about why each page exists
- Mention user benefit before technical detail
- Move from problem to solution to result
- Keep the pace steady so the presentation fits in 5 minutes
